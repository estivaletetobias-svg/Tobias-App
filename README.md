# IA Personal Coach | Aura System

Este repositório contém o código-fonte proprietário do ecossistema de performance humana **IA Personal Coach**, desenvolvido para o Tobias Estivalete.

## 🔗 Repositório Oficial
**GitHub:** [https://github.com/estivaletetobias-svg/Tobias-App.git](https://github.com/estivaletetobias-svg/Tobias-App.git)

## 🚀 Tecnologias
- **Frontend:** HTML5, CSS3 (Vanilla Aura Design System), Javascript (ES6+).
- **Backend (Visão):** Node.js + Assistants API (OpenAI).
- **Design:** Glassmorphism, Mesh Gradients, Mobile-First.

## 🏛️ Arquitetura do Produto
O app foca na "Presença Assíncrona Escalável", utilizando um sistema de **Master Brain** via API que gerencia múltiplos alunos através de injeção de contexto dinâmico.
